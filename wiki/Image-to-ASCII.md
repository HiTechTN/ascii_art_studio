# Image to ASCII Conversion

The Image to ASCII feature converts any image into ASCII art representation.

## How to Use

1. **Select an Image**
   - Tap "Pick Image" to open your photo gallery
   - Or use your camera to take a new photo

2. **Adjust Settings**
   - **Width**: Slide to adjust resolution (30-250 characters)
   - **Character Set**: Choose from 7 different character sets
   - **Color**: Select from 10 color palettes
   - **Invert**: Toggle to invert the output colors

3. **Advanced Options** (tap expand icon)
   - **Brightness**: Adjust the overall brightness
   - **Contrast**: Adjust the contrast level

4. **Export**
   - Copy to clipboard
   - Save as TXT file
   - Save as HTML (with colors)

## Character Sets

| Name | Description | Characters |
|------|-------------|------------|
| Standard | Classic ASCII art characters | ` .:-=+*#%@` |
| Detailed | More characters for better detail | Full ASCII set |
| Minimal | Simple characters | ` .:oO@` |
| Binary | Only 0 and 1 | `01` |
| Dots | Braille-style dots | `⠁⠂⠄⠆...` |
| Blocks | Block characters | `▤▥▦▧▨...` |
| Circles | Circle characters | `●◐◑◒◓...` |

## Tips for Best Results

1. **Choose high contrast images** - Images with clear contrasts work best
2. **Adjust brightness/contrast** - Fine-tune to get the best output
3. **Start with standard characters** - Then experiment with others
4. **Use appropriate width** - More characters = more detail but smaller text
