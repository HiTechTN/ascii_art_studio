# Frequently Asked Questions

## General

### What is ASCII Art Studio?
ASCII Art Studio is a cross-platform application that converts images to ASCII art and generates stylized ASCII text banners.

### What platforms are supported?
- Android (APK)
- iOS
- Web
- Windows
- macOS- Linux

### Is it free?
Yes! ASCII Art Studio is open source and free to use.

## Features

### How do I convert an image to ASCII art?
1. Go to the "Image" tab
2. Tap "Pick Image" to select a photo
3. Adjust settings as needed
4. Copy, save, or share your ASCII art

### How do I create text banners?
1. Go to the "Text" tab
2. Enter your text
3. Choose a font style
4. Select a color
5. Tap "Generate"

### How many character sets are available?
There are 7 character sets:
- Standard
- Detailed
- Minimal
- Binary
- Dots
- Blocks
- Circles

### How many color palettes are available?
There are 10 color palettes:
Neon Green, Amber, Cyan, Pink, Matrix Green, White, Purple, Orange, Blue, Red

### Can I export as HTML?
Yes! Image exports support HTML format with colors.

## Troubleshooting

### The image doesn't convert properly
- Try adjusting the brightness/contrast
- Ensure the image is not too complex
- Try a different character set

### The text is too small/large
- Adjust the width slider for image conversion
- For text banners, keep input shorter

### The app crashes
- Make sure you have the latest version
- Clear app cache and restart
- Reinstall if the problem persists

### Can I use images from my camera?
Yes, you can select images from your camera or gallery.
