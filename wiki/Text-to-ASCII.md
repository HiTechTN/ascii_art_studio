# Text to ASCII Generator

Create beautiful ASCII text banners from your custom text.

## How to Use

1. **Enter Text**
   - Type your message in the text input field
   - Press Enter or tap "Generate"

2. **Choose Font Style**
   - Block (5 lines) - Solid block letters
   - Banner (3 lines) - Classic banner style
   - Standard (5 lines) - Standard ASCII letters
   - Slant (5 lines) - Slanted/italic style
   - Lean (5 lines) - Leaning letters
   - Script (3 lines) - Cursive style
   - Big (7 lines) - Large letters

3. **Select Color**
   - Choose from 10 color palettes
   - Applies to the ASCII preview

4. **Export**
   - Copy to clipboard
   - Save as TXT file

## Supported Characters

The generator supports:
- Uppercase letters (A-Z)
- Numbers (0-9)
- Basic punctuation

## Tips

- Keep text under 20 characters for best results
- Longer text may not fit on smaller screens
- Use the "Big" font for maximum impact
