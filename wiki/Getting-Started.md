# Getting Started with ASCII Art Studio

Welcome to ASCII Art Studio! This guide will help you get started with the application.

## Installation

### Android (APK)
1. Download the latest APK from the [Releases](https://github.com/HiTechTN/ascii_art_studio/releases) page
2. Enable "Install from unknown sources" in your device settings
3. Open the APK file and install

### iOS
1. Clone the repository: `git clone https://github.com/HiTechTN/ascii_art_studio.git`
2. Run `flutter pub get`
3. Open in Xcode and build for your device

### Web
Visit the [GitHub Pages](https://hitechtn.github.io/ascii_art_studio/) version or build from source.

### Desktop (Windows/macOS/Linux)
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter build windows`, `flutter build macos`, or `flutter build linux`

## First Steps

### Convert an Image to ASCII
1. Launch the app
2. Tap "Pick Image" to select a photo
3. Adjust settings (width, character set, colors)
4. Copy, save, or share your ASCII art!

### Generate ASCII Text
1. Go to the "Text" tab
2. Enter your text
3. Choose a font style
4. Select your preferred color
5. Generate and export!

## Quick Tips
- Use higher resolution (more characters) for better detail
- Try different character sets for unique effects
- Use the brightness/contrast sliders to adjust the output
- Save your favorite creations to the Gallery
